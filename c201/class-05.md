## Chapter 5: “Images”

***To add an image into the page you need to use an `<img>` element.that must carry the following two attributes:***

- *`src` This tells the browser where it can find the image file.*
- *`alt` This provides a text description of the image which describes the image if you cannot see it.*
- *`title` You can also use the title attribute with the <img> element to provide additional information about the image.*
- *`height` This specifies the height of the image in pixels.*
- *`width`  This specifies the width of the image in pixels.*
- *`align `attribute was commonly used to indicate how the other parts of a page should flow around an image.*


*`<figure>` element used to contain images and their caption so that the two are associated `<figcaption>`*


## Chapter 11: “Color”

***The color property allows you to specify the color of text inside an element. You can specify any color in CSS in one of three ways:***
- *RGB Values (These express colors in terms of how much red, green and blue are used to make it up. For example: rgb(100,100,90))*
- *Hex Codes (These are six-digit codes that represent the amount of red, green and blue in a color, preceded by a pound or hash # sign. For example: #ee3e80)*
- *Color Names (There are 147 predefined color names that are recognized by browsers. For example: DarkCyan)*

*Opacity :CSS3 introduces the opacity property which allows you to specify the opacity of an element and any of its child elements.The value is a number between 0.0 and 1.0*

*hsl, hsla CSS3 introduces an entirely new and intuitive way to specify colors using hue, saturation, and lightness values.*


## Chapter 12: “Text”

***There are properties to control the choice of font, size, weight, style, and spacing.***

*The `font-family` property allows you to specify the typeface*

*The `font-size` property enables you to specify a size for the font by:*
- pixels (px)
- percentages (%)
- ems (em) size of text relative to the size of the text in the parent element.

*`@font-face` allows you to use a font, even if it is not installed on the computer of the person browsing, by allowing you to specify a path to a copy of the font, which will be downloaded if it is not on the user's machine.*

*`font-weight` property allows you to create bold text (ex: **bold**)*

`font-style` (ex:normal, _italic_, oblique)

*The `text-transform` property is used to change the case of text (uppercase,lowercase,capitalize)*

*`line-height` property sets the height of an entire line of text*

*`text-align` The text-align property allows you to control the alignment of text*
